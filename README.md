## おしゃべりうさこ部とは

「おしゃべりうさこ部」は、Discord上で3人のキャラクターが会話し、
その様子をWebで観測・再解釈できるプロダクトです。
キャラクター同士の関係性や会話の雰囲気を可視化し、
創作や分析の入口として楽しめることを目指しています。

## テーマ「さん」について
「3」人
キャラクター(○○「さん」)
拡「散」

### できること

- 3キャラクター（うさこ・ねここ・けろこ）がDiscordで会話
- 会話ログや日次レポートの閲覧
- キャラクター別の発言傾向やテーマの観測

### プロダクト構成

- Discord Bot: 3キャラクターの会話を生成・運用
- Web UI: 会話のログやレポートを見て楽しむ観測画面

### ねらい

- 会話を「物語」として楽しめる
- キャラクター理解や創作のヒントになる
- 観測・再解釈・共有の流れを作る

## 開発の進め方（重要）

### 基本ルール

* **apps/web は UI 専用**
* ロジック・データ解釈は `packages/` 側に寄せる予定
* Web 側では「表示」「観測」「操作」に集中する

---

### 触っていい場所

✅ OK

* `apps/web/src/components`
* `apps/web/src/pages`
* `apps/web/src/styles`
* 表示用のダミーデータ追加

❌ NG（相談してから）

* `packages/*`
* データ構造を変える大きな変更
* API仕様を決め打ちする変更
